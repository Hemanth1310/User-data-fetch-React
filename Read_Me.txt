
This Project is a Simple ReactJS Project which demonstrates the following

	Creating a Component in React
	Making HTTP calls
	Communicating between parent and child component
	Using Basic Routing in React

Device Side Setup
		
		npm start

Server Side Setup

	Default is set to Postman-mock api server
		(Just run the app)

    Else if		

	Creating Mock Api through json-server
	(Use the TestJSON.json file in the repo/ open Secound terminal in VSCODE and run following)
		commands:
			(@location of json file)
			install json-server:
				
				npm install json-server

			start a server with json files:

				json-server --watch TestJSON.json --port 3001



Prerequisites

Install Node JS
Refer to https://nodejs.org/en/ to install nodejs

Install create-react-app
Install create-react-app npm package globally. This will help to easily run the project and also build the source files easily. Use the following command to install create-react-app

npm install -g create-react-app
